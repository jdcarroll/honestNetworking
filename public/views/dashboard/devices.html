Device Discovery Timeline

  • Ping Subnet

    Sending broadcast signal to the network and verifying who responds. All devices on the system will be defualt respond with there mac address and ip address.

  • Grab Data From Arp Table

    When a broadcast signal gets sent accross the network the devices that see this this broadcast signal respond with there mac address and ip address. The device sending the broadcast signal then takes the response and stores that data in what is called and arp table. This process pulls that data from the table annd parses it making the data usable for that system

  • Nmap Scan

    This takes the computer object pulled from the arp table and runs a system scan on each ipadress. This scan will reveal the hostname of th device along with all open ports found on that sytem.
